# StealthBurner_Toolhead-Sensor_switch
ERCF Toolhead filament sensor without hall sensor. using magnetic and switch.

Preparing parts
- 6x3 neodymium magnetic 1EA
- switch 1EA
- M2*8mm 1EA
- M2*10mm 1EA
